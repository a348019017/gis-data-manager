GIS Data Manager

基于 Tauri 2 的轻量桌面 GIS 数据管理工具。统一管理多源数据、一键导入空间文件、注册地图服务、AI 智能辅助分析。

下载 v1.0.0-alpha GitHub

核心功能

覆盖 GIS 数据工作全流程

数据源管理

统一管理 PostgreSQL/PostGIS、MySQL、SpatiaLite 数据库，以及阿里云 OSS、AWS S3、MinIO 对象存储，支持一键连接测试。

PostgreSQL MySQL OSS/S3 SpatiaLite

数据导入与管理

基于 geozero 引擎，支持 Shapefile、GeoJSON、FlatGeobuf 一键导入 PostGIS。导入历史可追溯，支持标签分类与数据字典管理。

Shapefile GeoJSON FlatGeobuf 标签管理

服务注册与预览

注册 WMTS、WMS、WFS、TMS、GeoServer、ArcGIS REST 等 OGC 服务，内置 OpenLayers 地图实时预览。

WMTS WMS WFS GeoServer

GIS 工具箱

内置常用 GIS 工具：缓冲区分析、裁剪、相交、合并、坐标转换、空间连接、GDAL 栅格处理、NDVI 计算等。

空间分析 坐标转换 GDAL NDVI

AI 智能助手

接入 OpenAI / Claude / Ollama 等大模型，支持文件拖拽上传、上下文对话，辅助 GIS 数据分析与问题解答。

GPT-4o Claude Ollama 拖拽上传

轻量跨平台

Tauri 2 构建，安装包 <10MB。支持 Windows / macOS / Linux，系统托盘常驻，响应式布局适配小窗口。

Tauri 2 <10MB 响应式 系统托盘
